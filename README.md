# 📋 Smart To-Do List Application

A beginner-friendly Python project that helps users manage daily tasks efficiently.

---

## 🚀 Features

- 📅 Display current date and day
- ➕ Add new tasks
- ⏰ Set task date and time
- ⭐ Set task priority (High, Medium, Low)
- 📋 View all tasks
- ⏳ View pending tasks
- ✅ Mark tasks as completed
- 📦 Archive completed tasks
- 🔍 Search tasks
- ✏️ Edit existing tasks
- 🗓️ Monthly calendar view
- 🔔 Desktop reminder notification (15 minutes before task)
- 💾 Save tasks using JSON
- 🎨 Colorful terminal interface

## 📦 Installation

Clone the repository

```bash
git clone <repository-link>
```

Move into the project folder

```bash
cd Smart_ToDo_List
```

Install required libraries

```bash
pip install -r requirements.txt
```

Run the application

```bash
python main.py
```

---

## 📖 How to Use

1. Run the application.
2. Add a new task.
3. Set task date and time.
4. Assign a priority.
5. View, edit, or search tasks.
6. Mark completed tasks.
7. Receive reminder notifications before scheduled tasks.
